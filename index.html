Wee AI - Free Online AI Tools

Features

What is Wee AI? Wee AI is a website that provides free online AI tools. The AI tools provided for free are Chat AI and Text-to-Image Generator with various models available.

Chat AI

Make conversation with AI

Chat AI. Let AI assist your task with a prompt in conversation.

Image Generator AI

Make your image with AI

Image Generator AI. Let AI generate your image with a prompt.

Technologies

Wee AI uses the API provided by Huggingface to provide image creation services. Huggingface API provides many categories of AI Models. The model category used in this application is Text-to-Image.

Wee AI uses the API provided by Deepinfra to provide Chat AI services. Deepinfra provides many AI Chat Models, in this case, Wee AI takes all the AI Chat models available on Deepinfra to provide AI Chat services.

Wee AI create on Vite with React TS template and hosted in Firebase Hosting.
